Mobile Weather App

A quick and clean weather app for mobile view only. All weather data is pulled from the Open Weather API using Axios. Current functionalities include searching cities for their weather and tempurature. The app also features a history bar which the user can click on to remove or click to make that previous weather the main display. Currently working on a desktop only app with more functionality and stylings.

Code used was HTML CSS SCSS/SASS and React.
